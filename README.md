# 📦 Stock Hub

**Kelola stok dari Telegram, sinkronkan ke Google Sheets, lalu export ke marketplace hanya dalam beberapa klik.**

Stock Hub adalah Telegram Bot untuk membantu mengelola stok multi marketplace menggunakan Google Sheets sebagai database utama.

Bot ini dirancang untuk seller yang ingin mengurangi pekerjaan manual saat mengelola produk, stok, dan pesanan di Shopee maupun TikTok Shop.

---

## ✨ Fitur

### 📦 Manajemen Stok
- Cek stock
- Set stock
- Restock
- Sales (mengurangi stock)
- Update manual maupun upload Excel

### 🛍️ Produk
- Import produk dari Shopee
- Import produk dari TikTok Shop
- Update status produk
- Sinkronisasi data produk

### 📤 Marketplace
- Export format Shopee
- Export format TikTok Shop
- Export semua marketplace sekaligus

### 📊 Database
Menggunakan Google Sheets sebagai pusat data sehingga mudah dipantau dan diedit kapan saja.

### 🔒 Keamanan
- Duplicate order protection
- Activity log
- Queue processing untuk menjaga konsistensi data
- Auto backup data

### ⚡ Performa
Dirancang untuk menangani ribuan SKU dalam satu database Google Sheets.

**Pengujian internal:**
- ±5.000 produk
- Sinkronisasi stok Import & export marketplace

---

## 🚀 Cara Kerja

Semua proses dilakukan langsung melalui Telegram dengan command-command tertentu yang telah disediakan.

Dalam hitungan detik, Bot akan memproses file secara otomatis.
